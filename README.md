# ambiloop

Browser-based ambient audio layer composer. Layer, pitch-shift, and EQ audio files into seamless meditation loops. Pure client-side JavaScript.

## Features (V1)

- Load multiple MP3/WAV/OGG files (drag & drop or file picker)
- Arrange audio as layers on a timeline
- Per-layer pitch shifting (semitones, preserve duration via SoundTouchJS)
- Per-layer volume control
- Preview playback in real-time
- Export mixed audio as WAV

## Usage

Open `index.html` in a browser, or deploy to any static host. No build step, no server.
